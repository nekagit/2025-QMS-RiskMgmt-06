# Risk Management System - 2025-QMS-RiskMgmt-06

A Quality Management System application for identifying, assessing, and mitigating quality risks.

## Features

- Register risks with probability and impact assessment
- Automatic risk level calculation
- Category classification
- Mitigation strategy tracking
- Risk status management
- Risk register listing and deletion

## Tech Stack

- **HTMX** - Dynamic HTML interactions
- **SCSS** - Styling
- **TypeScript** - Type-safe JavaScript
- **Express.js** - Server framework

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Usage

1. Start the server: `npm run dev`
2. Open your browser to `http://localhost:3006`
3. Register risks using the form
4. View and manage risks in the register
